Информация о мульти-просмотре

1. Сохранение наборов камер мультипросмотра
2. Загрузка наборов камер из списка
3. Переключение качества камеры и протоколов в наборе камер.
4. Добавление второго потока к камере видеонаблюдения.
5. Элементы меню и архива в мультипросмотре.

1. Сохранение набора камер.

Для сохранения набор камер для мультипросмотра, Вам необходимо заполнить ячейки с элементами видео вашими камера. Далее выбрать в панели управления мультипросмотром ( ) элемент списка ( ). Введите название для группы камер мультипросмотра.

2. Включение сохраненного набора.

Для просмотра сохраненного набора камер мультипросмотра вам необходимо в панели управления мультипросмотром выбрать иконку загрузки ( ) и далее выбрать нужный вам элемент списка наборов камер.

3. Переключение качества камеры и протоколов в наборе камер.

В зависимости от количества камер в наборе формируется нагрузка браузера на ваш компьютер. Основной поток с камеры доступен только при просмотре 1-ой или 2-ух камер видеонаблюдения на экране. Для большего количества камер на экране отображается второй поток или последние кадры с камеры видеонаблюдения. Нагрузка при просмотре второго потока с камеры значительно ниже, чем просмотр кадров.
При включении в полноэкранном режиме видео поток переключается на максимальное качество. Рекомедуемые протоколы MSE или RTMP. Они передают видео в режиме реального время. Задержка в отображение регламентируется только скоростью загрузки камеры на вашем ПК.
Просмотр архива полностью доступен в резиме мультипросмотра, но отображает видео максимального качества. В связи с этим сильно возрастает нагрузка на Ваш ПК

4. Добавление второго потока к камере видеонаблюдения.

Самым эффективным способом использования мультипросмотра является использование второго потока с камеры видеонаблюдения. Включение второго потока не предусмотрено стандартным тарифным планом. Для включения второго потока используйте магазин дополнительных услуг или позвоните Вашему менеджеру для включения данной функции.

5. Элементы меню и архива в мультипросмотре.

Для вашего удобства и эффективного использования пространства элементы главного меню и таймлайн архива при влючении мультипросмотра сворачиваются. Для разворачивания элементов меню нажмите на самый нижний элемент списка меню. Для разворачивания таймлайна архива нажмите стрелку вверх посередини серой панели внизу экрана.
